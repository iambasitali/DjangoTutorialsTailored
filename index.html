Installing Django

Intro to Django:-

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel. It’s free and open source.

Today, Django stands as one of the most used web development frameworks. Developers widely prefer Django web development for its “batteries included” philosophy. It means that as a Django developer, you don’t have to install separate libraries for adding functionalities. Django provides a plethora of features to add standard functionalities in any kind of web application.

Quick install Django:-

Before you can use Django, you’ll need to get it installed. We have a complete installation guide that covers all the possibilities; this guide will guide you to a minimal installation that’ll work while you walk through the introduction.

Python:-

Being a Python Web framework, Django requires Python. See What Python version can I use with Django? for details. Python includes a lightweight database called SQLite so you won’t need to set up a database just yet. Get the latest version of Python at https://www.python.org/downloads/ or with your operating system’s package manager.

You can verify that Python is installed by typing python from your shell; you should see something like: 

                                
                                Python 3.x.y
                                [GCC 4.x] on linux
                                Type "help", "copyright", "credits" or "license" for more information.
                                >>>

Install Django:-

You've got two ways to install django on your system:

1: Install the latest release. For that run the following command: 

                                
                                pip install django

2: Install specific version of django. For example if you want to install release version 2.2.0 run the following command: 

                                
                                pip install django==2.2.0

Verify:-

To verify that Django can be seen by Python, type python from your shell. Then at the Python prompt, try to import Django:

                                
                                >>> import django
                                >>> print(django.get_version())
                                3.2
