Django Model Form:

As we discussed in last lesson django forms are much like django models. As we’re creating a database-driven app, chances are high to have forms that map closely to Django models.
In our case, we have a Category model, and you want to create a form that let user submit category. So it is unnecessary to define the field types in Category form, because we’ve already defined the fields in Category model.

So using Django model, we can create a Form using built-in helper class ModelForm. It will take Django Model as argument and convert it into Django Form. Let's define a new model named Alert in models.py using the below code:

todo_app/models.py

from django.db import models

class Alert(models.Model):
    title = models.CharField(max_length=200)
    description = models.TextField()
    is_delete = models.BooleanField(default=False)
    created_at = models.DateTimeField(auto_now_add=True,null=True)
    modified_at = models.DateTimeField(auto_now_add=True,null=True)

    def __str__(self):
        return self.title

To create a form for Alert model, dive into todo_app/forms.py and enter following code:

todo_app/forms.py

from .models import Alert
from django.forms import ModelForm


class AlertAddForm(ModelForm):
    class Meta:
        model = Alert # This will bind the ModelForm base class with Alert Model.
        fields = '__all__' # This will tell which field to be added in form.
        # Here we are using all fields that are present in model, but you can define a list of field names
        # Like ['Name', 'pub_date']
        # You can also define which fields needed to be excluded from form that is present in model.
        # Todo so we need to add a list of fields name in exclude attribute.
        # exclude = list of fields to be excluded

In above example, we have imported ModelForm from django.forms and create a class that inherits from ModelForm.

Finallay, we need to create a view that will save the form directly to the database through rendering the form. In todo_app/views.py add the below code:

todo_app/views.py
def addAlert(request):
    if request.method == 'POST':
        form = AlertAddForm(request.POST)
        if form.valid(): # This will check if the fields data provided by user is  valid and according to the fields that we defined in Form definition class.
            form.save(commit=False)
        return redirect('index')
    else:
        form = AlertAddForm()
        return render(request, 'todo_app/add_alert.html', {'form': form})

Lets create a URLconf for Alert form that maps the view to a route path. Open todo_app/urls.py and append following code inside urlpatterns list: 

todo_app/urls.py
path('add-alert', views.addAlert, name='addAlert'),
