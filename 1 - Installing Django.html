Installing Django

Django is a python framework which is used to do rapid development, developers can build apps with in very short time Moreover Its free and open source. At that time Django is very modern and most used web development framework, It provides maximum supports to the third party libraries.

Quick install Django

Before you can use Django, you’ll need to get it installed. We have a complete installation guide that covers all the possibilities; this guide will lead you to a minimal installation that’ll work while you walk through the introduction.

Python:

Before starting work we needs to install Python in our system, we can download the python from the below link https://www.py thon.org/downloads/ After installing the Python, we need to check whether the Python is installed or not with the below command. 

$ python --version

If this will return you the Python version it means python is installed otherwise in case of error python is not installed.

C:\> python --version
Python 3.9.6

This is the version which I am currently using in my local PC.

Note
On Mac or Linux you have to use python3 --version.

Python2 or Python3?

Currently python is upto 3+ and I will prefer to use Python 3.7 because it is very latest version of python.

Virtual Envirnment:

The virtual environment is an environment which is used by Django to run an application. This is very good approach to create and execute a Django application in a separate environment. Python provides a tool virtualenv to create an isolated Python environment. We will use this to create a virtual environment for our Django application.

Below are the steps to install and activate a virtual envirnment

$ python -m venv venv

By running this command the below folder structure will be created

/venv
  Lib
  Scripts
  pyvenv.cfg

You can navigate to venv folder with the below commands. This command cd {folder-name} used to switch to the child directory.

$ cd venv

Below is the command to switch back to parent folder.

$ cd ..

Below is the command to activate the virtual envirnment before installing Django. We have to just go to Script folder and run the below command.

$ \venv\Scripts> activate

When you will activate venv it will look like as mentioned below

(venv) $ \venv\Scripts>
Note
On Mac or Linux you have to use "source venv/bin/activate" to activate virtual environment.

Install Django:

You've got two ways to install django on your system.

1: Install the latest release. For that run the following command:

(venv) $ pip install django

2: Install specific version of django. we will work on 3.1.3 version of Django for these tutorials.

(venv) $ pip install django==3.1.3
Important!
For this tutorial make sure you install Django version 3.1.3 otherwise your application may not work properly.

Note
On Mac or Linux instead of pip you have to use pip3.

Verify Django Installation:

To verify that Django can be seen by Python, type python from your shell. Then at the Python prompt, try to import Django:

(venv) $ python -m django --version

If it will show the Django version as mentioned below, it means Django is installed successfully otherwise Djanog is not installed.

(venv) $ python -m django --version
3.1.3
Note
On Mac or Linux instead of python you have to use python3.
