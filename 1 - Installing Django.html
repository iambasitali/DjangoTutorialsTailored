Installing Django

Django is a python framework which is used to do rapid development, developers can build apps with in very short time Moreover Its free and open source. At that time Django is very modern and most used web development framework, It provides maximum supports to the third party libraries.

Quick install Django

Before you can use Django, you’ll need to get it installed. We have a complete installation guide that covers all the possibilities; this guide will guide you to a minimal installation that’ll work while you walk through the introduction.

Python:

Before starting work we needs to install Python in our system, we can download the python from the below link https://www.py thon.org/downloads/ After installing the Python, we need to check whether the Python is installed or not with the below command. 

python --version

Install Django:

You've got two ways to install django on your system.

1: Install the latest release. For that run the following command:

pip install django

2: Install specific version of django. For example if you want to install release version 2.2.0 run the following command:

pip install django==2.2.0

Verify:

To verify that Django can be seen by Python, type python from your shell. Then at the Python prompt, try to import Django:

django --version
